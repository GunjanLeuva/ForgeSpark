# ForgeSpark 🚀

🧩 **Novice-level Atlassian Forge App** built for the **Forge Quest Hackathon**.

## 📋 What it Does

- Greets the current user by name (e.g., Hello Gunjan!)
- Displays the current Confluence space name
- Shows current theme (light/dark mode) using Forge APIs

## 🧠 What I Learned

- How to build UI using Forge React components
- Fetch user and theme context in a Confluence app
- Use Forge bridge to connect frontend and backend

## 🌟 What's Next

- Add richer UI with Forge UI Kit 2
- Extend to Jira panels and third-party API integrations
